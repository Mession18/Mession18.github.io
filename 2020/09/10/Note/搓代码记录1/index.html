P2181 对角线

题目描述

对于一个 n 个顶点的凸多边形，它的任何三条对角线都不会交于一点。请求出图形中对角线交点的个数。

例如，6 边形：

输入格式

输入只有一行一个整数 n，代表边数。

输出格式

输出一行一个整数代表答案。

输入输出样例

输入 #1

输出 #1

输入 #2

输出 #2

说明/提示

数据规模与约定

  • 对于 50 %50% 的数据，保证 3≤n≤1003 \leq n \leq 1003≤n≤100

  • 对于 100 %100% 的数据，保证 3≤n≤1053 \leq n \leq 10^53≤n≤105

解题

解题思路（其实是照学别人的数学太鸡儿差了）
∵没有3条对角线同时相交的情况

∴一个交点只有2条对角线，2条对角线也就是确定了4个顶点。

然后就是组合(第一次取可以ｎ个点都是可以取的，第二次取的时候第一个取的点就不能取了，所以只能取(n-1)种，以此类推)

再然后因为四个顶点顺序交换也不会改变对角线。所以要除以4！就是24

数学式子就是n∗(n−1)∗(n−2)∗(n−3)/24n*(n-1)*(n-2)*(n-3)/24n∗(n−1)∗(n−2)∗(n−3)/24

但听说直接除以24容易暴毙（运算过多？

所以就把24分成2，3，4往前分上去

就有了n∗(n−1)/2∗(n−2)/3∗(n−3)/4n*(n-1)/2*(n-2)/3*(n-3)/4n∗(n−1)/2∗(n−2)/3∗(n−3)/4
