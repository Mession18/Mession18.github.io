其中Q1Q_1Q1​为热机从高温热源吸热的热量,Q2Q_2Q2​为热机向低温热源放热的热量(的绝对值).由于循环过程ΔEint=0\Delta E_{int} = 0ΔEint​=0,故做功W=Q1−Q2W = Q_1 - Q_2W=Q1​−Q2​

制冷机

应用逆循环工作的热机称为制冷机

制冷系数:

e=Q2W=Q2Q1−Q2e = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2} e=WQ2​​=Q1​−Q2​Q2​​

其中Q1Q_1Q1​为制冷机从高温热源吸热的热量,Q2Q_2Q2​为制冷机向低温热源放热的热量

卡诺循环

卡诺循环是在两个温度恒定的热源之间工作的循环过程.

卡诺循环的效率:

η卡=1−T2T1\eta_{卡} = 1 - \frac{T_2}{T_1} η卡​=1−T1​T2​​

T1T_1T1​为高温热源温度,T2T_2T2​为低温热源温度.

热力学第二定律

如果一个过程发生在封闭系统中,对于不可逆过程,系统的熵增加;对于可逆过程,系统的熵不变.系统的熵永不减小

γ=11−v2c2\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}} γ=1−c2v2​​1​

其中ccc为光速, vvv为两个参照系的相对速度

x′=γ(x−vt)y′=yz′=zt′=γ(t−vc2x)x' = \gamma(x - vt)\\ y' = y\\ z' = z\\ t' = \gamma(t - \frac{v}{c^2}x) x′=γ(x−vt)y′=yz′=zt′=γ(t−c2v​x)
